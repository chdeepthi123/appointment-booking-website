# Appointment Booking Website

## Project Overview
This project is a Responsive Appointment Booking Website developed using HTML, CSS, and JavaScript.  

The goal of this assignment was to build a clean, responsive, and interactive frontend application without using any frameworks or CSS libraries.

---

## Setup Instructions

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any web browser.
4. Navigate to the booking page and test the appointment booking functionality.

No additional installation or dependencies are required.

---

## Approach

The project was developed using:

- HTML5 for structure and semantic layout.
- CSS3 (Flexbox & Media Queries) for responsive design.
-  JavaScript for:
  - Time slot selection logic
  - Form validation
  - Booking summary display
  - LocalStorage data storage (Bonus feature)

The layout was designed mobile-first and adjusted using responsive breakpoints to ensure compatibility across mobile, tablet, and desktop devices.

The code is structured into separate folders for better organization:
- HTML files for pages
- CSS folder for styling
- JS folder for JavaScript logic

---

## Assumptions Made

- The appointment booking is simulated (no backend integration).
- Only one time slot can be selected per booking.
- Users will enter valid phone numbers.
- Data is stored locally in the browser using LocalStorage.
